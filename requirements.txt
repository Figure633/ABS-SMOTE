## Usage

A basic example of applying ABS-SMOTE to an imbalanced dataset is shown
below:

```python
from sampling.abs_smote import AdaptiveBoundaryShiftSMOTE

sampler = AdaptiveBoundaryShiftSMOTE(
    k=5,
    n_samples=1.0,
    alpha=0.5,
    random_state=42
)

X_resampled, y_resampled = sampler.fit_resample(X, y)
